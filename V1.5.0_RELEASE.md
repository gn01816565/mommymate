# MommyMate v1.5.0 - 保險規劃指南 🛡️

## 🎉 重大更新

### 全新功能：保險規劃指南

為準媽媽提供**完整的保險知識與規劃工具**，幫助您了解孕期保險需求、試算保費、規劃預算。

---

## ✨ 功能特色

### 1️⃣ 保險總覽
- **優先順序清楚標示**
  - ⭐⭐⭐⭐⭐ 婦嬰險（最重要！）
  - ⭐⭐⭐⭐⭐ 實支實付醫療險
  - ⭐⭐⭐⭐ 住院日額險
  - ⭐⭐⭐ 癌症險 / 重疾險

- **重要提醒**
  - 婦嬰險必須懷孕前投保
  - 等待期 30-90 天
  - 懷孕後無法加保

- **政府補助說明**
  - 14 次免費產檢
  - 各縣市生育補助
  - 育嬰留職停薪津貼
  - 托育補助

### 2️⃣ 保費試算器 💰
**互動式試算工具**
- 輸入年齡（18-50 歲）
- 勾選需要的保險類別
- 即時計算年繳 / 月繳保費

**自動調整**
- 35 歲以上保費 +20%
- 40 歲以上保費 +50%

**試算範例**
- 30 歲孕婦，完整組合：約 18,000-35,000 元/年
- 每月約 1,500-2,900 元

### 3️⃣ 投保檢查清單 ✅
**6 大重點項目：**
1. ✅ 確認是否已懷孕
2. ✅ 檢視現有保單
3. ✅ 比較多家保險公司
4. ✅ 仔細閱讀保單條款
5. ✅ 了解等待期規定
6. ✅ 確認保障範圍

**進度條視覺化**
- 勾選完成項目
- 即時顯示完成度
- 彩色進度條

### 4️⃣ 保險名詞解釋 📖
**8 個常用名詞：**
- 婦嬰險（孕婦險）
- 實支實付醫療險
- 妊娠併發症
- 等待期
- 除外責任
- 失能險（殘扶險）
- 理賠比例
- 誠實告知義務

**淺顯易懂**
- 簡單說明
- 實例解釋
- 注意事項標示

## 📍 使用方式

### 如何開啟
1. 前往 https://showgan.com/mommymate/
2. 選擇任一懷孕週數
3. 切換到「檢查與補助」標籤
4. 點擊「🛡️ 保險規劃指南」卡片

### 建議使用時機
- **懷孕前** - 規劃保險需求
- **懷孕初期** - 檢視現有保單
- **隨時** - 了解保險知識

---

## 💡 實際應用

### 案例：30 歲準媽媽
**需求：** 第一胎，無現有保單

**使用流程：**
1. 查看保險總覽 → 了解優先順序
2. 使用保費試算器 → 估算預算
3. 勾選投保檢查清單 → 確保完整規劃
4. 查詢名詞解釋 → 理解專業術語

**試算結果：**
- 婦嬰險：3,500 元/年
- 實支實付：7,500 元/年
- 住院日額：4,000 元/年
- **總計：15,000 元/年**（約 1,250 元/月）

## ⚠️ 重要提醒

### 法律聲明
**以上資訊僅供參考**
- 實際保障與費率請洽專業保險顧問
- 各保險公司條款可能不同
- 建議多方比較後決定

### 投保建議
1. **及早規劃** - 懷孕前投保最佳
2. **多方比較** - 至少 3 家保險公司
3. **仔細閱讀** - 保單條款與除外責任
4. **誠實告知** - 健康狀況與懷孕情形
